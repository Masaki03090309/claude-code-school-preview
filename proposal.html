作成目的: 個人向けスクールのコース設計叩き台
対象: 株式会社新大陸 提案用

※本資料の価格・期間・サポート範囲は仮設定です。
※本資料では「Cloud Code」ではなく、Anthropicの Claude Code（クロードコード） を前提にしています。
※法人向けAI研修・助成金活用プランは別商品として設計し、本資料は主に個人向けB2Cスクールの設計を対象としています。

1. 企画コンセプト

このスクールは、単なる「Claude Codeの使い方講座」ではなく、非エンジニアが業務改善人材から案件対応人材へ成長し、最終的に新大陸の案件・業務委託・採用へ接続できるキャリア直結型スクールとして設計する。

新大陸の強みは、学習の先に

  • 新大陸の既存顧客へのAI活用支援
  • FDE（伴走型AI実装支援）
  • SES・業務委託案件
  • 新大陸本体または提携先への採用

という出口を設計できる点にある。
そのため、商品としての見せ方も「AIを学ぶ」ではなく、AIで業務を変え、仕事につなげることを主語にする。

2. 企画の全体設計

2-1. 4プランの役割整理

プラン 役割 主な対象 到達イメージ
初級プラン 入口商品 未経験・非エンジニア AIで小さな業務改善ツールを1本作れる
中級プラン 主力商品 副業予備軍・社内DX担当候補 業務アプリを1本作り、運用まで説明できる
実践プラン 利益商品 案件獲得・転職を狙う層 提案・実装・納品まで一通り回せる
Proプラン 選抜商品 上位受講生・採用候補 新大陸案件に入れるFDE/Claude Code実務人材になる

2-2. 共通の提供形式

全プラン共通で、以下の形式を前提にする。

  • 動画学習（eラーニング）
  • コースごとの課題提出
  • グループレビューまたはワークショップ
  • 伴走コーチによる質問対応
  • テンプレート配布（CLAUDE.md、要件定義、提案書、README等）
  • 月次アップデート補講（Claude Codeの最新機能反映）

2-3. この設計にする理由

  • 非エンジニア層は、毎週固定時刻で集まる形式より、自分のペースで進められる動画学習 + 添削 + 伴走の方が継続しやすい
  • Claude Codeは変化が速く、教材を一度作って終わりではなく、月次で最新化する仕組みが必要
  • 新大陸の強みは「教えること」単体ではなく、案件や採用に接続できることなので、全コースで成果物と選抜導線を持たせる必要がある

3. 仮の価格設計

3-1. 販売価格（仮）

プラン 想定期間 通常価格（税込・仮） 初期モニター価格（税込・仮） 位置づけ
初級プラン 1ヶ月 88,000円 69,800円 最初の成功体験を作る入口商品
中級プラン 2ヶ月 248,000円 198,000円 もっとも売上の中心になる主力商品
実践プラン 3ヶ月 498,000円 398,000円 案件化・転職導線を担う利益商品
Proプラン 3ヶ月 798,000円 598,000円 選抜制・配属/採用直結商品

3-2. 価格設定の考え方

今回の仮価格は、以下の市場レンジを踏まえて設定している。

  • 低価格帯: Udemy型の単発講座
  • 中価格帯: DMM 生成AI CAMPやSHIFT AIのようなコミュニティ・学び放題型
  • 高価格帯: TEKIONのような少人数伴走・実装型ブートキャンプ

新大陸案は、単なる動画配信ではなく、

  • 課題添削
  • 実務テンプレ配布
  • 伴走コーチ
  • 実案件接続の可能性
  • 採用・業務委託への導線

を持つため、コミュニティ型よりは高く、法人向け少人数ブートキャンプよりは入りやすい価格帯に置くのが妥当と判断した。

3-3. 販売上の補足

  • 初級は、入口として価格ハードルを下げる
  • 中級は、最も受講者数を取りにいく中心商品にする
  • 実践は、成果物と案件化価値が高いため単価を上げる
  • Proは、少人数・選抜制・レビュー負荷が高いため高単価に設定する
  • 将来的には「継続割」「一括申込割」「採用トラック特典」を追加してもよい

4. 初級プラン｜AI業務改善スターター

4-1. プラン概要

  • 想定期間: 1ヶ月
  • 想定学習時間: 20〜30時間
  • 通常価格: 88,000円（税込・仮）
  • 初期モニター価格: 69,800円（税込・仮）

4-2. ペルソナ

  • 非エンジニアの会社員
  • 営業、事務、バックオフィス、建設事務、現場管理補助など
  • AIチャットは使ったことがあるが、自分でツールやアプリを作ったことはない
  • 将来の副業・転職・昇給に少し興味がある

4-3. ゴール

Claude CodeとVS Codeの基本環境を整え、テンプレートを活用しながら、業務改善ミニツールを1本完成・公開し、3分で説明できる状態になる。

4-4. ベネフィット

  • AIを「質問するだけ」から「成果物を作る」側に移れる
  • 非エンジニアでも、自分に実装ができるという成功体験が得られる
  • 次の中級プランに進むための土台ができる

4-5. コース構成

コース1｜Claude Codeの全体像と環境構築

  • Claude Codeの役割とできること
  • VS Codeの基本操作
  • ターミナル、ファイル、フォルダの理解
  • GitHubアカウント準備と基本的な使い方
  • 最初のプロジェクト立ち上げ

コース2｜非エンジニア向け実装の基礎

  • HTML / CSS / JavaScriptの最低限の理解
  • Claude Codeへの依頼の出し方
  • エラーの読み方と直し方
  • コードを壊したときの戻し方
  • 小さく試す・確認する・修正する流れ

コース3｜業務改善ミニツール制作

以下のような題材から1つ選んで制作する。

  • 議事録整理ツール
  • 見積書ドラフト補助ツール
  • CSV/スプレッドシート整理ツール
  • 資料・写真整理ツール

コース4｜公開・説明・発信

  • READMEの作り方
  • デモの見せ方
  • 導入前後の変化の言語化
  • 3分デモ動画の作成
  • 成果物をポートフォリオとして見せる方法

4-6. 成果物

  • 業務改善ミニツール 1本
  • README 1本
  • 3分デモ動画 1本
  • 業務改善レポート 1本

4-7. 修了後の出口

  • 中級プランへの継続受講
  • 社内の小さな業務改善提案
  • AI活用実績としての初期ポートフォリオ化

4-8. この価格・内容にする理由

初級の目的は、難しい理論を教えることではなく、最短で成功体験を作ることにある。
そのため、価格も高すぎず、しかし「安い動画教材」で終わらない水準に設定する。
新大陸の将来売上は中級・実践・Proで作るため、初級は利益最大化よりも入口としての強さを重視する。

  • 初級修了者、または同等の基礎力がある人

5-3. ゴール

DB・認証・ファイル管理・API連携を含む業務アプリを1本作り、要件と運用まで説明できる状態になる。

  • 実践プランへの進学条件を満たしやすくなる

5-9. この価格・内容にする理由

6-8. この価格・内容にする理由

そのため、提案・要件定義・納品まで含めて価値が大きく、初級・中級より高単価にしてよい。
また、新大陸の事業構造においては、この層が増えるほどFDE・SES・受託開発との接続がしやすくなるため、事業上の重要度が高いプランとなる。

7. Proプラン｜配属・採用直結プログラム

7-1. プラン概要

  • 想定期間: 3ヶ月
  • 想定学習時間: 90〜120時間
  • 通常価格: 798,000円（税込・仮）
  • 初期モニター価格: 598,000円（税込・仮）
  • 参加条件: 選抜制 / 面談制

7-2. ペルソナ

  • 実践プラン上位修了者
  • 新大陸案件に入りたい人
  • 業務委託や正社員候補として見られたい人
  • 顧客対応、提案、運用まで見られる人材を目指す人

7-3. ゴール

顧客業務を診断し、PoCを設計し、チーム設定・セキュリティ・運用まで見られるFDE/Claude Code実務人材になる。

7-4. ベネフィット

  • 学習者ではなく、実務人材として見られる状態に近づく
  • 新大陸案件、業務委託、採用への接続可能性が高まる
  • 高単価案件や上位キャリアへの移行がしやすくなる

7-5. コース構成

コース1｜顧客ヒアリングと業務マップ作成

  • 顧客課題の聞き出し方
  • 業務フローの整理
  • ボトルネックの特定
  • 関係者整理

コース2｜PoC企画とROI設計

  • PoCテーマの決め方
  • ROIの見せ方
  • 導入優先順位
  • 社内提案の組み立て方

  • 新大陸の業務委託・パートナー候補
  • 提携先企業への転職推薦候補
  • 上位案件へのアサイン候補

7-8. この価格・内容にする理由

Proは上級講座ではなく、配属前訓練・選抜プログラムである。
レビュー密度、少人数運営、案件・採用への接続設計を考えると、通常のスクールより高単価でも成立しやすい。
また、新大陸にとっては採用コストや育成コストを先に回収できる意味もあるため、事業性の観点でも高単価設計が妥当である。

8. 新大陸ならではの差別化ポイント

8-1. 学んだ先に「案件」がある

多くのAIスクールは、学習で終わる。
新大陸案は、学習 → 実践 → 案件候補化 → 業務委託/採用という導線が作れる点が最大の差別化になる。

8-2. 新大陸自身が需要側を持っている

新大陸は、既存顧客へのAI活用支援、FDE、受託開発、人材紹介へつながる可能性を持っている。
そのため、ただ学ぶだけでなく、実務に使われる人材へ育てる設計ができる。

8-3. 建設・中小企業・営業/バックオフィスの題材が作れる

一般的なスクールは、汎用テーマに寄りやすい。
新大陸は、見積書、資料管理、問い合わせ管理、営業進捗、写真/資料台帳など、既存顧客の課題に直結した教材テーマを持てる。

8-4. FDEメソッドを教えられる

新大陸案の本質は、ツール学習ではなく、

  • 業務棚卸し
  • 課題発見
  • AI化診断
  • MVP提案
  • 実装
  • 納品
  • 運用改善

まで含めた実装型の支援人材を育てることにある。

9. 販売設計のおすすめ方針

9-1. 初期フェーズ

最初は、以下の順番で立ち上げるのが良い。

  1. 初級・中級を先に販売開始する
  2. 事例が出た受講生をもとに実践プランへ送客する
  3. 実践プラン上位者だけをProに引き上げる
  4. Pro修了者を案件・採用へ接続する

9-2. マーケティング訴求

訴求は「Claude Codeを学ぶ」より、以下の表現がよい。

  • 非エンジニアでもAIで業務改善ツールが作れる
  • AIを武器に社内評価・昇給・副業に繋げる
  • 学んだ先に案件・業務委託・採用のチャンスがある
  • 仕事で使えるアプリを作れるようになる

9-3. 注意点

  • 「案件保証」「採用保証」は避ける
  • 表現は 評価基準達成者を案件候補・採用候補に接続 とする
  • 品質担保のため、Proは必ず選抜制にする
  • 教材は固定化せず、月次更新前提で運営する

  • ただし案件接続・採用接続の価値を乗せられる

11. 最終提案

新大陸が出すべき商品は、単なる「Claude Code講座」ではなく、非エンジニアがAIで業務アプリを作り、案件・業務委託・正社員採用まで接続できるキャリア直結型スクールである。

そのため、4プランは以下の役割で整理するのが最も筋が良い。

  • 初級: AIで小さな成果物を作れる人を増やす
  • 中級: 業務アプリを実装できる人を作る
  • 実践: 提案・納品までできる人を作る
  • Pro: 新大陸の案件に入れる人を選抜する

この設計であれば、他社の汎用AIスクールと真正面から価格競争せず、新大陸の顧客基盤・案件供給力・採用力を参入障壁として使える。
